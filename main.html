Mes projets

Projet 1 : Application Python de gestion de tâches

Ce projet consistait à développer une application en Python pour gérer les tâches quotidiennes. J'ai intégré des fonctionnalités d'ajout, suppression, et modification de tâches, ainsi que la possibilité de trier les tâches par priorité ou par date d'échéance. Ce projet m'a permis d'améliorer mes compétences en Python, et de me familiariser avec les concepts de gestion de base de données.

Projet 2 : Site web interactif avec JavaScript

J'ai créé un site web interactif utilisant HTML, CSS et JavaScript. Le site comprend une galerie d'images dynamique, des animations, et une fonctionnalité de formulaire de contact. Ce projet m'a permis de maîtriser les concepts de manipulation du DOM avec JavaScript, ainsi que d'améliorer mes compétences en responsive design pour assurer la compatibilité avec tous les appareils.

Projet 3 : Automatisation de tests logiciels

Dans ce projet, j'ai utilisé Python pour automatiser des tests de logiciels. J'ai mis en place des scripts de test automatisés pour vérifier la fonctionnalité d'une application avant son déploiement. Ce projet m'a apporté une grande expérience dans la gestion de la qualité logicielle, ainsi qu'une meilleure compréhension des méthodes de développement agile.
